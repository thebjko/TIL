# 2. `@property`

```python
class Foo:
    def __init__(self):
        self.__baz = 29
    
    @property
    def bar(self):
        return self.__baz

    @bar.setter
    def bar(self, new_value):
        self.__baz = new_value * 100
        return 13   # 가능?
```

```python
f = Foo()
f.bar   # 6
f.bar = f.bar = 20
f.bar # 2000
```

```python
f = Foo()
f.bar += f.bar
f.bar   # 1200
```

```python
print((f.bar:=20)) 에러 발생
```
