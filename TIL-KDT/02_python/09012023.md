## 튜플(Tuple)

- 변경 불가능하며, 반복 가능함
```python
a = (1, 2, 3, 4)
a[1] = '3'   # -> 불가능
```
- 순서를 가지며 서로 다른 타입의 요소를 가질 수 있음
- 불변한 값들의 나열
- 항상 소괄호 형태로 정의, 콤마로 구분
